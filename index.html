WeatherPy: Exploring Cities and Their Weather

This website is the project page for my Python API project repository.

I wanted to take a look at how various weather measurements changed depending on a city's location in the world. To begin, I took a look at OpenWeatherMap's Weather API, and structured my queries based on randomly generated latitudes and longitudes. Using a list of nearly 600 coordinate pairs, I automated my queries to the API in order to obtain weather information for the nearest city to those coordinates on February 5th, 2022. For details on how I automated my queries, please refer to the WeatherPy Jupyter Notebook in my repository.

Once I obtained a dataset I could use, I decided to look at four weather measurements: temperature, humidity, cloudiness, and wind speed. After obtaining these values, I wanted to know if a city's latitude had any correlation with those measurements.

This site takes a look at not only the plots of all cities, but also the plots of cities divided between the northern and southern hemisphere, and attempts to find if there are any correlations using linear regressions. All plots were made using Pandas and Matplotlib.

For access to the dataset, please visit the data page where you will also find a link to download the dataset as a CSV file.

This website was created using Minty Bootswatch made by Thomas Park which is built on top of Bootstrap 5 and uses Bootstrap 5's JavaScript libraries for added functionality.
